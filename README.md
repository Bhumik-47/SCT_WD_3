## Project Description

This is an interactive implementation of the classic two-player Tic-Tac-Toe game. This project focuses heavily on complex game logic, state management, and event handling within a browser environment.

## Key Features

* **Two-Player Mode:** Players take alternating turns (X and O) by clicking on the board cells.
* **Game State Management:** The state of the 3x3 board is tracked efficiently using a JavaScript array, allowing for quick checks against winning conditions.
* **Winning Logic:** Comprehensive logic checks all eight possible winning conditions (rows, columns, and diagonals) after every move.
* **Visual Feedback:** The status display guides the players by announcing the current player's turn, the winner, or a draw. Winning cells are visually highlighted (green background) when the game concludes.
* **Restart Functionality:** A single button resets the entire game state, clears the board, and prepares for a new round.

## Technologies Used

| Technology | Purpose |
| :--- | :--- |
| **HTML5** | Structuring the 3x3 game board using data attributes for index tracking. |
| **CSS3** | Utilizes CSS Grid for the perfect 3x3 board layout; styling for players X and O; and visual effects for win/loss states. |
| **JavaScript (ES6)** | Implementation of all game logic, turn management, state array manipulation, and result validation. |

## How to Run
